# 📄 PRD – Gemini 기반 개인 업무 비서 & 일정·메모 자동화 웹앱 (Demo)

## 1. 프로젝트 개요

### 1.1 프로젝트 명
Gemini 기반 개인 업무 비서 & 일정·메모 자동화 웹앱

### 1.2 프로젝트 목적
자연어로 입력한 사용자의 업무 요청을 Gemini API와 LangChain을 활용해 이해·분류하고, 일정 및 메모를 자동으로 생성·관리하는 **AI 업무 비서 데모 웹앱**을 개발한다.

본 프로젝트는 비전공자도 이해할 수 있는 구조로, **AI 활용 + 웹 서비스 기본 구조**를 학습하는 것을 목표로 한다.

### 1.3 대상 사용자
- IT 비전공자
- 업무 자동화·AI 비서에 관심 있는 학습자
- Next.js + 생성형 AI 기반 포트폴리오를 만들고자 하는 훈련생

---

## 2. 서비스 핵심 컨셉

> “사람의 말 → AI의 이해 → 규칙적인 처리 → 자동 저장”

### 서비스 한 줄 설명
사용자가 말로 업무를 요청하면, AI가 이해하여 일정·메모로 자동 정리해주는 개인 업무 비서 웹앱

---

## 3. 전체 시스템 흐름

1. 사용자가 자연어로 업무 요청 입력
2. Next.js 서버(API Route)가 요청 수신
3. Gemini API가 요청 의도와 핵심 정보 분석
4. LangChain이 업무 유형에 따라 처리 흐름 분기
5. 결과 데이터를 검증 후 DB에 저장
6. 저장된 일정·메모 데이터를 다시 조회 가능

---

## 4. 기술 구성요소 및 역할 (UI 제외)

### 4.1 Next.js (서버 역할)
- API Route를 통해 서버 기능 수행
- Gemini API 및 LangChain 호출 담당
- 클라이언트와 AI 처리 로직 사이의 중계 역할

> 비유: AI 비서에게 일을 전달하는 **접수 창구**

---

### 4.2 환경변수 관리 (.env)
- Gemini API Key 등 민감 정보 저장
- 서버에서만 접근 가능하도록 구성

> 비유: 중요한 열쇠를 보관하는 **금고**

---

### 4.3 Gemini API (자연어 처리 엔진)
- 사용자의 자연어 요청 이해
- 일정·메모 형태로 구조화된 데이터 생성

주요 기능:
- 문장 의도 파악(Intent 분석)
- 요약 및 정리
- 텍스트 생성

> 비유: 사람 말을 알아듣는 **AI 두뇌**

---

### 4.4 출력 포맷 고정 (JSON)
- AI 응답을 자유 텍스트가 아닌 JSON 형태로 유도
- 이후 자동 처리 및 DB 저장 가능

예시 개념:
- intent: create_event / create_memo
- title
- date
- time
- content

> 비유: 자유 글이 아닌 **작성 양식 제출**

---

### 4.5 LangChain (업무 자동화 흐름 관리)
- AI 처리 과정을 단계별로 구조화
- 업무 유형에 따른 분기 처리

주요 역할:
- Router: 일정 / 메모 / 요약 구분
- Chain: 이해 → 검증 → 저장 → 응답
- (선택) Memory: 최근 요청 맥락 유지

> 비유: AI에게 일을 시키는 **업무 매뉴얼**

---

### 4.6 데이터베이스(DB)

#### 목적
- 생성된 일정과 메모를 저장·관리

#### 최소 데이터 구조
- events (일정)
  - title
  - dateTime
  - description
- memos (메모)
  - content
  - tags
  - createdAt

> 비유: 비서의 **업무 수첩**

---

### 4.7 데이터 검증(Validation)

#### 목적
- AI 결과의 형식 오류 방지
- 잘못된 데이터 저장 예방

검증 항목 예시:
- 날짜 형식 검증
- 시간 형식 검증
- 필수 값 누락 여부

> 비유: 문서 저장 전 **서식 검사**

---

### 4.8 사용자 구분 (선택 기능)

- 개인 비서 컨셉을 위한 사용자 분리
- 데모 단계에서는 단일 사용자도 가능

확장 시:
- 간단한 로그인 기능
- 사용자별 데이터 분리

> 비유: 수첩의 **주인 확인**

---

### 4.9 업무 자동화(스케줄링) – 확장 기능

- 정해진 시간에 자동 요약 생성
- 오늘의 일정 정리 기능

예시:
- 매일 아침 일정 요약 생성
- 주간 할 일 자동 정리

> 비유: 비서가 먼저 일을 해두는 **자동 루틴**

---

## 5. 권장 개발 순서 (구현 로드맵)

1. Next.js API Route 생성

6. 데이터 검증 로직 추가
7. (선택) 로그인 기능 추가
8. (선택) 스케줄 기반 자동화 기능 확장
